# FAST BIZ

## 프로젝트 개요

- 취업 준비생과 주식 투자자가 기업에 대한 비재무적 정보를 쉽게 탐색 및 분석할 수 있도록 QA 기반 챗봇 서비스

## 사용법

- 백엔드 repository는 이 [링크](https://github.com/T0nixx/ssafy-seasonal-backend/)를 참조해주세요.

- 로컬 실행 방법<br>

  ```
  npm install
  ```

  ```
  npm run start
  ```
